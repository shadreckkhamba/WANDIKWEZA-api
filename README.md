# Documentation

## Overview
This API serves as an intermediary between the client-side API and the virtual server. Its primary purpose is to execute queries on the virtual server while ensuring that direct access to the server is restricted for security and control purposes.

## Features
- **Blueprints**: Blueprints will be created for each service that utilizes the data provided by the API.
- **Endpoints**: Specific endpoints will be developed for individual indicators, allowing targeted access to data.

To set up the API, follow the standard Python setup process. Ensure all dependencies are installed and configured correctly to enable smooth operation.
